🎯 ويدجت النموذج المتطور

📋 التعليمات:
  1. استبدل YOUR-WORKER-DOMAIN برابط الـ Worker الخاص بك
  2. انسخ الكود أدناه والصقه في موقعك
  3. يمكنك تخصيص التصميم حسب احتياجاتك

💡 مثال الاستخدام - كود HTML + JavaScript

⚙️ خيارات التخصيص

  • الألوان: غير متغيرات CSS مثل #3498db للون الأزرار
  • الخطوط: استبدل Arial, sans-serif بخط مختلف
  • الحجم: عدّل max-width: 600px لتغيير عرض النموذج
  • الرسائل: خصص نصوص الرسائل والتأكيدات
